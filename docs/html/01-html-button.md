# Balise Button

# Attribut
	Attribut        Utilité

	type            définit l’action du bouton (button / submit / reset)
	name            clé envoyée au backend (uniquement dans un form)
	value           valeur envoyée au backend (avec name)
	id              identifiant du bouton
	disabled        empêche l’action et le clic

# Exemple de base
	<button type="submit">Envoyer le formulaire</button>
	<br><br>
	<button type="reset">Réinitialiser</button>
	<br><br>
	<button type="button" name="action" value="delete">Supprimer</button>
