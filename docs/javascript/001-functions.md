
# Liste des Types de Functions

  Type de fonction	        Function type	            this	Usage typique

  Fonction déclarée	        Function declaration	    Propre (nouveau this)

  Fonction d’expression	    Function expression	      Propre (nouveau this)	

  Fonction fléchée	        Arrow function	          Hérité (pas de nouveau this)

  Méthode d’objet	          Object method             this = objet contenant	

  Fonction constructeur	    Constructor function	    this = instance créée	

  Méthode de classe	        Class method	            this = instance
