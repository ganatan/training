# Concept

  Un Observable est un objet qui représente un flux de données dans le temps.

  subscribe() = tu t’abonnes à la boîte aux lettres (Observable).
  Chaque fois qu’un message arrive → tu réagis (code dans subscribe).
  Quand tu n’en as plus besoin → tu dois te désabonner, sinon tu continues à recevoir du courrier.

  subscribe() = je m’abonne
  unsubscribe() = j’arrête de recevoir des messages
