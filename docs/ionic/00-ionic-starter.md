# Concepts
  ionic n’est pas une alternative à Angular SSR.
  Angular SSR est une app web servie depuis un VPS.
  Ionic est une app mobile native embarquant Angular.


  Ionic
   Compilation Angular dans une app mobile native via Capacitor
   App qui tourne dans une WebView avec accès aux API du téléphone.

# Interets
  Une application mobile installable (Android / iOS)
  Avec accès natif (caméra, GPS, notifications, stockage, etc.)
  En réutilisant Angular comme moteur d’UI et de logique.

  les composants <ion-*> pensés “UX mobile” (listes, tabs, toasts, modals, etc.)
  la navigation mobile (stack, transitions, gestures)
  Capacitor pour packager et accéder au natif
  le build Android/iOS
