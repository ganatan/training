# Opérateur ternaire en TypeScript / JavaScript
**Conditional (ternary) operator**

## Définition
  L’opérateur ternaire est une expression conditionnelle compacte qui remplace une structure `if / else`.

## Syntaxe générale
  `condition ? valeur_si_vrai : valeur_si_faux`

## Objectif
  Évaluer une condition et retourner une valeur différente selon le résultat, le tout en **une seule ligne**, sans bloc `if / else`.

## Exemple
`result = x > 10 ? 10 : x`
