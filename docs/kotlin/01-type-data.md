# Type de Data


Kotlin	            Description	Exemple
String	            chaîne de caractères	        "Hello"
Int	                entier 32 bits	                42
Long	            grand entier 64 bits	        4000000000L
Double	            flottant 64 bits	            3.14
Float	            flottant 32 bits	            3.14F
Boolean	            booléen	                        true
Char	            caractère	                    'A'

Array<String>	    tableau typé	                arrayOf("A","B")
List<String>	    liste immuable	                listOf("A","B")
